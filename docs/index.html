Classic mode for Wikipedia

How to Get the Old Wikipedia Layout Back, without creating an account

In January 2023, the Wikimedia Foundation started rolling out a new design on the English version of Wikipedia, as well as on its sister projects. For various reasons, many Wikipedia readers and editors expressed their frustration with that. Fortunately, there are a lot of options to get the old interface back, without having to create an account on Wikipedia.

I've been developing a solution to this problem, initially for my own use, under the form of a Firefox add-on, "Classic mode for Wikipedia". You can very easily install it by following this link. You can also check other users' reviews before choosing to install it.

How to use it? Just install the add-on, and any Wikipedia page you visit will now have the "classic" look predating 2023.

This is a non-commercial add-on, free, open-source, and without advertisement (and it will stay this way). The add-on also works with some other popular Wikimedia websites: Wikimedia Commons, Wiktionary, Wikidata, Wikisource, and MediaWiki.

In order to work, when installing it, the add-on will ask your permission to access your data for Wikipedia and Wikimedia websites. Below is a screenshot of what the request pop-up should look like. Simply click the "Add" button the first time you install the add-on:

Nothing to worry about, the add-on does not collect or send your data anywhere, it just needs to know when you are accessing a Wikipedia page, in order to offer you automatically the classic interface.

Below you will find additional information about it in case you are interested (features, contact information, privacy, license, etc.). I hope you will find this add-on useful!

In case this add-on is not exactly what you're after, I listed a couple of other similar free and functional add-ons by other developers. The list is available just here.

Features

  • It works automatically on all versions of Wikipedia, no matter the language. No complicated configuration required, simply install the add-on and it works straight away!
  • It works on several other Wikimedia projects: Wikimedia Commons, Wiktionary, Wikisource, Wikidata, MediaWiki.
  • It respects your privacy. It is free, with no ads, no tracking, no data collection. It just offers you Wikipedia with its classic layout, nothing more, nothing less.
  • Easy uninstall: if you want the new Wikipedia/Wikimedia layout back, simply disable or uninstall the add-on.
  • The code is published under a free, open-source license, and is publicly reviewable by anyone.
  • It works with all versions of Firefox published after 2017.
  • Contrary to many other similar add-ons, it leaves Nostalgia Wikipedia untouched. Nostalgia Wikipedia is a read-only copy of the English Wikipedia from December 20, 2001, with an even older layout than the "classic" one.

Privacy and license

The add-on is non-commercial, and does not collect or send any data anywhere (if it were the case, as a resident of the European Union, I would be forced by law to tell you that it collects your data).

The add-on is published under the Mozilla Public License Version 2.0, which is a free, open-source license, guaranteeing that anyone can read and reuse freely the source code at any time.

The code itself is pretty simple, and any developer or code-savvy people can easily check its content, safety, and features.

Alternatives

Besides Classic mode for Wikipedia there are many other add-ons for Firefox or other browsers to get the old Wikipedia interface back.

Here is a non-exhaustive list, as of January 24, 2023 (these add-ons may have been updated since):

  • Old Wiki. Supports many (all?) Wikimedia projects: Wikipedia, Wikimedia Commons, Wiktionary, Wikitravel, etc. It uses a very clean and safe code. Unfortunately it is not available for Firefox yet, but it supports many other browsers (Chrome, Edge, Brave and Opera). Personal rating: 4.5/5
  • Wikipedia Vector Skin. Quite popular add-on. It uses a code similar to Classic mode for Wikipedia, and is also available for Firefox only. Unfortunately its scope is limited to wikipedia.org and mediawiki.org (e.g. it does not include Wikimedia Commons, Wiktionary, etc.), but it does its job very well with a clean and efficient code, and respects your privacy. Personal rating: 4/5
  • wikivector. For Firefox only. Scope limited to wikipedia.org. It does its job, but in a less efficient way than other add-ons. Personal rating: 2.5/5.
  • legacy skin wiki. Same shortcomings than wikivector. Personal rating: 2.5/5

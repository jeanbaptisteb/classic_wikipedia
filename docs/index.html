A Firefox add-on to Get the Old Wikipedia Layout Back
(without creating an account or other tedious configuration)

The Wikimedia Foundation decided recently (January 2023) to roll out a new design on the English version of Wikipedia, called "Vector 2022". If you don't like it, you can simply fix the issue by installing this add-on I developed for Firefox: "Classic mode for Wikipedia".

The add-on is completely free and open-source, without advertisement, data collection, or anything like that -and it will stay this way. You can read some user reviews right here.

How does it work? Just install the add-on, and any Wikipedia page you visit will now have the "classic" look predating 2023. No need to create an account. (If you're keen for more technical details, you can check its source code, or write me an email with your questions.)

The add-on will also work with some other popular Wikimedia projects, in case the Vector 2022 layout were eventually deployed on them: Wikimedia Commons, Wiktionary, Wikidata, Wikisource, and MediaWiki.

If this add-on is not exactly what you're after, I listed a couple of similar free and functional add-ons created by other developers. The list is available just below. But I hope you will find my add-on useful! In any case, feel free to leave some general feedback on the review page.

If you have a more detailed feedback you want to share (bug report, feature request, etc), besides emailing me, you can write a comment on the dedicated thread on Mozilla forums, or directly file a bug report or feature request on Github if you're familiar with that.

Alternatives

Besides "Classic mode for Wikipedia", there are many other add-ons for Firefox or other browsers to get the old Wikipedia interface back.

Here is a non-exhaustive list, as of January 25, 2023 (these add-ons may have been updated since):

  • Old Wiki. Supports many (all?) Wikimedia projects: Wikipedia, Wikimedia Commons, Wiktionary, Wikitravel, etc. It uses a very clean and safe code. Unfortunately it is not available for Firefox yet, but it supports many other browsers (Chrome, Edge, Brave and Opera). Personal rating: 4.5/5
  • Wikipedia Vector Skin. Quite popular add-on. It uses a code similar to Classic mode for Wikipedia, and is also available for Firefox only. Unfortunately its scope is limited to wikipedia.org and mediawiki.org (e.g. it does not include Wikimedia Commons, Wiktionary, etc.), but it does its job very well with a clean and efficient code, and respects your privacy. Personal rating: 4.5/5
  • wikivector. For Firefox only. Scope limited to wikipedia.org. It does its job, but in a less efficient way than other add-ons. Personal rating: 2.5/5.
  • legacy skin wiki. Same shortcomings than wikivector. Personal rating: 2.5/5
